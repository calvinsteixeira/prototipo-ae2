# Tela de Cadastro

## Descrição

Permite criar uma nova conta na plataforma.

## Campos

- Foto de perfil
- Nome completo
- Email
- Telefone
- Endereço
- Senha
- Confirmar senha

## Botões

- `Cadastrar`: envia os dados e cria a conta.

## Funcionalidade

- Verificação de campos obrigatórios.
- Validação de email e confirmação de senha.
- Em caso de sucesso, redireciona para o login.
